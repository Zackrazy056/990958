# 数值复现总指南

## 目标

这份指南把当前工作区中的数值脚本和论文中的核心概念一一对应起来，帮助你把“看懂综述”推进到“亲手跑过关键图像和关键数量级”。

## 脚本与论文映射

### 1. `finite_string_modes.py`

对应：

- 第 1 轮
- 正常模、离散谱、封闭边界条件

输出：

- 前几个本征函数图
- 数值本征频率与解析结果对比

### 2. `compact_support_qnm_scan.py`

对应：

- 第 2 轮
- Laplace 变换、Green 函数、Wronskian、QNM 极点

输出：

- 紧支撑势图
- `log |W(s)|` 热图
- 候选极点位置

### 3. `compact_support_qnm_rootfind.py`

对应：

- 第 2 轮
- 真正的 `W(s)=0` 复根迭代

输出：

- 带根轨迹的 `log |W(s)|` 热图
- 复 Newton 迭代历史
- 精修后的 toy QNM 复根

说明：

- `compact_support_qnm_scan.py` 负责粗扫和找种子
- `compact_support_qnm_rootfind.py` 负责把种子升级成真正的复根

### 4. `compact_support_qnm_track.py`

对应：

- 第 2 轮之后的升级训练
- 参数连续变化下的 QNM 漂移

输出：

- `Re(s)` 随势高或势宽变化
- `Im(s)` 随势高或势宽变化
- 复平面上的根轨迹

说明：

- 这是从“单点求根”升级到“continuation 跟踪”
- 最适合回答“势改变时 QNM 会怎么移动”这类科研问题

### 5. `compact_support_qnm_map2d.py`

对应：

- 第 2 轮之后的二维升级
- 同时扫描 `height` 和 `width`

输出：

- 第一、第二 toy QNM 的 `Re(s)` 二维热图
- 第一、第二 toy QNM 的 `Im(s)` 二维热图

说明：

- 这一步把一维 continuation 推广成真正的二维参数空间图
- 最适合做“参数如何塑造谱结构”的可视化

### 6. `compact_support_root_vs_ringdown.py`

对应：

- 第 2 到 3 轮之间的桥接
- 把频域根和时域 ringdown 放在同一个参数扫描里比较

输出：

- `Re(s)`：频域根 vs 时域拟合
- `Im(s)`：频域根 vs 时域拟合
- 复平面轨迹对照
- 若干代表性波形与拟合窗口

说明：

- 这个脚本的重点是“趋势对应”
- 由于激发方式、有限拟合窗口和模态混合，时域拟合不一定逐点等于频域根
- 但它非常适合解释：参数变化会怎样同时推着根和 ringdown 变化

### 7. `time_domain_wave_evolution.py`

对应：

- 第 2 到 3 轮
- 初始 burst、ringdown、晚时间衰减

输出：

- 势图
- 观测点波形
- 半对数图

### 8. `tail_comparison.py`

对应：

- 第 3 轮
- 紧支撑势与长程势的晚时间差异

输出：

- 两类势的对比图
- 晚时间半对数信号对比

### 9. `schwarzschild_potentials.py`

对应：

- 第 4 轮
- Regge-Wheeler 与 Zerilli 势

输出：

- `V(r)` 对比
- `V(r_*)` 对比

### 10. `wkb_schwarzschild_qnm.py`

对应：

- 第 5 轮
- Schwarzschild 基模数量级、势峰近似、WKB

输出：

- 势峰位置
- 一阶 WKB 的 `M\omega`
- 与文献值对比

### 11. `black_hole_parameter_estimation.py`

对应：

- 第 12 轮
- 论文公式 (67) 和 (68)

输出：

- 频率与阻尼时间随自旋变化图
- 用合成观测值回收黑洞质量和自旋

### 12. `tov_polytrope.py`

对应：

- 第 7 轮
- TOV 背景、质量半径关系

输出：

- `M-R` 曲线
- 代表性背景模型剖面

### 13. `neutron_star_empirical_relations.py`

对应：

- 第 12 轮
- 论文公式 (69) 和 (70)

输出：

- `f` 模经验关系示意

### 14. `ringdown_fit_demo.py`

- 第 12 轮
- 探测、噪声、参数回收

输出：

- 含噪波形
- 拟合曲线
- 回收参数误差

```

```

## 对结果的正确期待

这些脚本分为两类：

- 教学脚本
  目标是展示结构和数量级，不追求论文级数值精度。
- 半定量脚本
  能给出合理数量级并与文献值做基础比较。

因此，最重要的检查点不是“最后一位小数是否一致”，而是：

- 频率尺度是否正确
- 图像结构是否正确
- 物理解释是否通顺
