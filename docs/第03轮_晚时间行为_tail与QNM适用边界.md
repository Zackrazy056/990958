# 第 03 轮：晚时间行为、tail 与 QNM 适用边界

## 对应原论文

- Section 2 末尾
- PDF 第 8 到 9 页

## 本轮目标

这一轮解决一个很容易被忽略、但对真正理解 QNM 非常关键的问题：

- QNM 能不能展开一切？
- QNM 主导的是哪一段时间？
- 为什么 Schwarzschild 这类系统最终还会出现 tail？

## 精讲

原论文在公式 (15) 和 (16) 的讨论里非常谨慎。它没有说“真实解等于所有 QNM 的完备展开”，而是说：

- 在固定的有限空间区间里
- 在足够晚的时间里
- 真实解可以被一组极点贡献很好地近似

这意味着 QNM 是一种局部的、渐近的、晚时间主导描述，而不是自动等价于严格的完备基底。

最重要的物理图景是三段式：

1. `burst`
   初始数据的直接传播和散射响应。
2. `ringdown`
   一小组阻尼最慢的 QNM 主导信号。
3. `tail`
   更晚时间的慢衰减背景响应，常常与 branch cut 或长程势相关。

对紧支撑势，QNM 图景最干净，晚时间可以是指数衰减主导。  
对 Schwarzschild 这类在无穷远慢衰减的势，branch cut 会引入幂律尾波，因此“真正无限晚”的主导者不再是单纯的离散 QNM。

## 推导与理解

### 1. 为什么极点项先出现

逆 Laplace 变换沿 Bromwich 轮廓积分：

\[
\chi(t,x)=\frac{1}{2\pi i}\int_{\Gamma} e^{st}\hat{\chi}(s,x)\,ds.
\]

将积分轮廓向左变形时：

- 会扫过离散极点
- 每个极点给一个留数项
- 这些项构成阻尼振荡

因为阻尼最慢的模有最大的 `Re(s)`，所以在中间晚时间，主导项必然来自前几个 QNM。

### 2. 为什么 tail 会更晚接管

如果 Green 函数只有孤立极点，晚时间结构会更接近离散指数和。  
但当势在无穷远不够快地衰减时，解析延拓后会出现 branch cut。

branch cut 的积分贡献虽然在中间时间往往不显眼，但它可能衰减得比某些指数模式更慢，于是在“真正无限晚”的极限中反而统治波形。

这就是论文为什么提醒你：QNM 很重要，但不是全部。

## 数值训练

### 训练 A：时域演化识别三阶段

对一个局域势做时域演化，在固定观测点画出：

- 线性坐标下的波形
- `log |psi|` 随时间变化

你应标出：

- 初始到达时间
- ringdown 拟合窗口
- 更晚阶段是否偏离单一指数

### 训练 B：比较紧支撑势与慢衰减势

做两次演化：

1. 紧支撑势，例如方势垒或紧支撑平滑 bump
2. 慢衰减势，例如模仿 Schwarzschild 外区 `V \sim 1/x^2`

对比：

- 谁的 ringdown 更“纯”
- 谁更容易出现长尾

## 易错点

- 不要把“QNM 可观测”误解成“QNM 严格完备”。
- 不要把“尾波更晚接管”误解成“QNM 不重要”；实际观测窗口里 QNM 往往最关键。
- 不要忽略空间区间限制。原论文强调的是有限区间中的晚时间，而不是整个空间上的全局展开。

## 本轮结论

QNM 控制的是开放系统波形中的 ringdown 段，它们来自 Green 函数极点；而更晚时间的 tail 往往与 branch cut 和长程势有关。因此，QNM 是核心，但不是全部。
